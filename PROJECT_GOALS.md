# Project Goals

1. **Hybrid AI:** Integrate both local and cloud-based AI models to provide flexible and powerful capabilities.

2. **Memory System:** Develop a persistent memory system that allows the AI to learn from past interactions and retain context across sessions.

3. **Payment Integration:** Implement a secure payment system to handle transactions for premium features or API usage.

4. **VS Code Extension:** Create a Visual Studio Code extension for seamless integration with the developer's workflow.
